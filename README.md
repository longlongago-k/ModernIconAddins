# ModernIconAddins
Modern Icon Office-VSTO Addins and useful tools.  
モダンアイコンをOfficeアプリなどに活用するツールです。

## 機能 (Features)
現在以下3つのツールが提供されます。

- Excel用アドイン Excel addin
- PowerPoint用アドイン PowerPoin addin
- アイコン画像取得用スタンドアロンツール(.exe) StandAlone tool
